\section{Conclusões}
	Pode-se concluir que:
	\begin{itemize}
		\item O valor de alpha não é irrelevante para o tempo de execução, mas o número de iterações se é muito importante
		\item Ao ter partes aleatorias o algoritmo não sempre dá uma solução muito ótima 
		\item Os valores ótimos para el calculo de custo com percurso total dos elevadores são $\alpha = 0.45$ e número de iterações igual a 100 % CAMBIAR
		\item Os valores ótimos para el calculo de custo com tempo de espera e tempo no elevador são $\alpha = 0.25$ e número de iterações igual a 100 % CAMBIAR
		\item Os valores ótimos para el calculo de custo com ambos critérios anteriores são $\alpha = 0.4$ e número de iterações igual a 100 % CAMBIAR
	\end{itemize}
