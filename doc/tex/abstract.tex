\abstract{
	Neste trabalho o objetivo foi explorar o problema Partial Weighted Max-SAT com diferentes parámetros e comparar seus resultados, para isso foi usado solvers oficiais de \cite{MaxSAT14}.
A primeira parte do trabalho consiste em fazer um gerador de dados com o padrão oficial wcnf (weighted conjunction normal form). As seguintes partes são experimentos feitos para o tamanho de clausulas dois e tres variando seus parámetros como o número de clausulas, tamanho de clausulas e número de átomos.
A continuação presentamos os experimentos, resultados e conclusões.
}
